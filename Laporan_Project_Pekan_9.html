BAB 1 - PENDAHULUAN

1.1 Latar Belakang

Dalam dunia pengembangan perangkat lunak modern, penggunaan version control system (VCS) merupakan hal yang sangat penting dan fundamental. Git adalah salah satu version control system yang paling populer dan banyak digunakan oleh developer di seluruh dunia. GitHub, sebagai platform hosting untuk repository Git, menyediakan layanan kolaborasi dan penyimpanan kode secara cloud yang memudahkan pengembang dalam mengelola proyek perangkat lunak mereka.

Project Pekan 9 ini bertujuan untuk mempraktikkan pembuatan source code sederhana berupa program "Hello World", kemudian melakukan commit dan push kode tersebut ke akun GitHub. Melalui project ini, mahasiswa diharapkan dapat memahami alur kerja dasar dalam penggunaan Git dan GitHub untuk manajemen source code.

1.2 Tujuan

Tujuan dari project ini adalah:

  1. Memahami konsep dasar version control system (Git)
  2. Mampu membuat project source code sederhana (Hello World)
  3. Mampu melakukan inisialisasi repository Git pada project lokal
  4. Mampu melakukan commit perubahan kode ke repository lokal
  5. Mampu melakukan push kode dari repository lokal ke GitHub
  6. Memahami alur kerja pengembangan perangkat lunak menggunakan Git dan GitHub

BAB 2 - KEBUTUHAN / REQUIREMENT

Sebelum memulai project, terdapat beberapa kebutuhan perangkat lunak dan akun yang harus disiapkan terlebih dahulu. Berikut adalah daftar requirement yang diperlukan:

No Kebutuhan Deskripsi Status

2.1 Instalasi Visual Studio Code

Visual Studio Code (VS Code) adalah code editor gratis dan open-source yang dikembangkan oleh Microsoft. VS Code mendukung berbagai bahasa pemrograman dan memiliki banyak extension yang berguna untuk pengembangan perangkat lunak.

Langkah-langkah instalasi VS Code:
  1. Buka browser dan kunjungi https://code.visualstudio.com/
  2. Klik tombol "Download for Windows"
  3. Jalankan file installer yang telah diunduh
  4. Ikuti wizard instalasi hingga selesai
  5. Buka VS Code untuk memastikan instalasi berhasil
📸 [Screenshot: Tampilan VS Code setelah instalasi berhasil]

2.2 Instalasi Git

Git adalah version control system terdistribusi yang digunakan untuk melacak perubahan pada source code selama pengembangan perangkat lunak. Git memungkinkan beberapa developer bekerja secara bersamaan pada project yang sama.

Langkah-langkah instalasi Git:
  1. Buka browser dan kunjungi https://git-scm.com/downloads
  2. Pilih versi yang sesuai dengan sistem operasi (Windows)
  3. Unduh dan jalankan file installer
  4. Pada proses instalasi, gunakan pengaturan default (Next > Next > Install)
  5. Verifikasi instalasi dengan membuka Command Prompt/Terminal dan ketik:
$ git --version git version 2.45.0.windows.1
📸 [Screenshot: Hasil perintah git --version di terminal]

2.3 Membuat Akun GitHub

GitHub adalah platform web-based untuk hosting repository Git. GitHub memungkinkan developer untuk menyimpan, mengelola, dan berbagi kode mereka secara online.

Langkah-langkah membuat akun GitHub:
  1. Buka browser dan kunjungi https://github.com/
  2. Klik tombol "Sign up"
  3. Masukkan email, password, dan username yang diinginkan
  4. Verifikasi akun melalui email yang dikirimkan
  5. Login ke akun GitHub yang telah dibuat

BAB 3 - INSTRUKSI PROJECT

3.1 Membuat Project Source Code Hello World

Pada tahap ini, kita akan membuat project Hello World menggunakan HTML, CSS, dan JavaScript. Project ini terdiri dari beberapa file yang saling berkaitan untuk menghasilkan tampilan halaman web yang menampilkan pesan "Hello, World!".

3.1.1 Membuat Folder Project

Langkah pertama adalah membuat folder untuk menyimpan file-file project. Buka Terminal/Command Prompt dan jalankan perintah berikut:

$ mkdir hello-world-project $ cd hello-world-project
📸 [Screenshot: Pembuatan folder project di terminal]

3.1.2 Membuat File index.html

File index.html adalah file utama yang berisi struktur HTML dari halaman web Hello World. File ini juga memuat styling CSS inline dan me-referensikan file JavaScript eksternal.

Buka VS Code, buat file baru bernama index.html, dan tulis kode berikut:

<!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Hello World - Project Pekan 9</title> <style> * { margin: 0; padding: 0; box-sizing: border-box; } body { font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; display: flex; justify-content: center; align-items: center; min-height: 100vh; background: linear-gradient(135deg, #0f0c29, #302b63, #24243e); color: #ffffff; } .container { text-align: center; padding: 40px 60px; background: rgba(255, 255, 255, 0.05); border-radius: 20px; backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.1); box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3); animation: fadeIn 1.5s ease-out; } h1 { font-size: 3.5rem; background: linear-gradient(90deg, #f093fb, #f5576c, #4facfe); -webkit-background-clip: text; -webkit-text-fill-color: transparent; } @keyframes fadeIn { from { opacity: 0; transform: translateY(30px); } to { opacity: 1; transform: translateY(0); } } </style> </head> <body> <div class="container"> <h1 id="greeting">Hello, World!</h1> <p id="message">Selamat datang di Project Pekan 9</p> </div> <script src="script.js"></script> </body> </html>

Penjelasan kode HTML:

  • <!DOCTYPE html> — Mendeklarasikan tipe dokumen sebagai HTML5
  • <html lang="id"> — Tag pembuka HTML dengan atribut bahasa Indonesia
  • <head> — Bagian header yang berisi metadata, title, dan CSS
  • <style> — CSS untuk styling tampilan dengan efek glassmorphism dan gradient
  • <body> — Bagian body yang berisi konten utama halaman
  • <h1>Hello, World!</h1> — Heading utama yang menampilkan pesan "Hello, World!"
  • <script src="script.js"> — Me-referensikan file JavaScript eksternal
📸 [Screenshot: Kode index.html di VS Code]

3.1.3 Membuat File script.js

File script.js berisi logika JavaScript yang menambahkan fungsionalitas interaktif pada halaman web, seperti menampilkan waktu saat ini dan efek klik pada heading.

// Hello World - Project Pekan 9 // Script JavaScript untuk menampilkan pesan Hello World function sayHello() { console.log("Hello, World!"); console.log("Project Pekan 9 - Hello World berhasil dijalankan!"); } function showCurrentTime() { const now = new Date(); const options = { weekday: 'long', year: 'numeric', month: 'long', day: 'numeric', hour: '2-digit', minute: '2-digit', second: '2-digit' }; const timeString = now.toLocaleDateString('id-ID', options); const messageEl = document.getElementById('message'); if (messageEl) { messageEl.textContent = `Selamat datang! Hari ini: ${timeString}`; } } document.addEventListener('DOMContentLoaded', function() { sayHello(); showCurrentTime(); });

Penjelasan kode JavaScript:

  • sayHello() — Fungsi untuk mencetak "Hello, World!" ke console browser
  • showCurrentTime() — Fungsi untuk menampilkan tanggal dan waktu saat ini dalam format Indonesia
  • DOMContentLoaded — Event listener yang menjalankan fungsi-fungsi saat halaman selesai dimuat
📸 [Screenshot: Kode script.js di VS Code]

3.2 Inisialisasi Git Repository

Setelah source code Hello World selesai dibuat, langkah selanjutnya adalah menginisialisasi Git repository pada folder project. Proses ini membuat folder project menjadi sebuah Git repository yang dapat melacak setiap perubahan file.

3.2.1 Konfigurasi Git

Sebelum menggunakan Git, kita perlu mengkonfigurasi nama dan email yang akan digunakan untuk setiap commit. Buka terminal dan jalankan perintah berikut:

$ git config --global user.name "Nama Anda" $ git config --global user.email "email@example.com"

Penjelasan:

  • git config --global user.name — Mengatur nama pengguna Git secara global
  • git config --global user.email — Mengatur email pengguna Git secara global

3.2.2 Inisialisasi Repository

Buka terminal pada folder project hello-world-project dan jalankan perintah berikut:

$ cd hello-world-project $ git init

Perintah git init akan membuat subdirektori .git di dalam folder project. Subdirektori ini berisi semua file yang diperlukan oleh Git untuk melacak perubahan.

Initialized empty Git repository in C:/Users/[username]/hello-world-project/.git/
📸 [Screenshot: Hasil perintah git init di terminal]

3.2.3 Menambahkan File ke Staging Area

Setelah repository diinisialisasi, kita perlu menambahkan semua file ke staging area menggunakan perintah git add:

$ git add . $ git status

Perintah git add . menambahkan semua file dalam direktori saat ini ke staging area. Perintah git status digunakan untuk melihat status file yang telah ditambahkan.

On branch main No commits yet Changes to be committed: (use "git rm --cached <file>..." to unstage) new file: .gitignore new file: README.md new file: index.html new file: script.js
📸 [Screenshot: Hasil perintah git status di terminal]

3.2.4 Melakukan Commit Pertama

Commit adalah proses menyimpan snapshot perubahan ke repository lokal. Jalankan perintah berikut untuk membuat commit pertama:

$ git commit -m "Initial commit: Hello World project"

Flag -m digunakan untuk menambahkan pesan commit secara langsung. Pesan commit harus deskriptif agar mudah dipahami.

[main (root-commit) abc1234] Initial commit: Hello World project 4 files changed, 120 insertions(+) create mode 100644 .gitignore create mode 100644 README.md create mode 100644 index.html create mode 100644 script.js
📸 [Screenshot: Hasil perintah git commit di terminal]

3.3 Push Project ke GitHub

Setelah melakukan commit di repository lokal, langkah selanjutnya adalah push (mengunggah) kode ke GitHub agar tersimpan secara online dan dapat diakses dari mana saja.

3.3.1 Membuat Repository Baru di GitHub

Langkah-langkah:
  1. Login ke akun GitHub di https://github.com
  2. Klik tombol "+" di pojok kanan atas, lalu pilih "New repository"
  3. Isi informasi repository:
    • Repository name: hello-world-project
    • Description: Project Hello World - Tugas Pekan 9
    • Visibility: Public
    • Jangan centang "Add a README file" (karena kita sudah punya)
  4. Klik tombol "Create repository"
📸 [Screenshot: Halaman pembuatan repository baru di GitHub]

3.3.2 Menghubungkan Repository Lokal dengan GitHub

Setelah repository dibuat di GitHub, kita perlu menghubungkan repository lokal dengan repository remote di GitHub. Jalankan perintah berikut di terminal:

$ git remote add origin https://github.com/[username]/hello-world-project.git $ git branch -M main

Penjelasan:

  • git remote add origin — Menambahkan remote repository dengan nama "origin" yang mengarah ke URL repository GitHub
  • git branch -M main — Mengubah nama branch utama menjadi "main"

3.3.3 Push ke GitHub

Langkah terakhir adalah melakukan push untuk mengunggah semua commit dari repository lokal ke GitHub:

$ git push -u origin main

Flag -u (upstream) digunakan untuk mengatur branch tracking sehingga pada push berikutnya cukup menggunakan git push saja.

📸 [Screenshot: Hasil perintah git push di terminal]

3.3.4 Verifikasi di GitHub

Setelah push berhasil, buka browser dan kunjungi repository GitHub Anda untuk memverifikasi bahwa semua file telah terunggah dengan benar.

Verifikasi:
  1. Buka https://github.com/[username]/hello-world-project
  2. Pastikan semua file (index.html, script.js, README.md, .gitignore) terlihat
  3. Periksa commit history untuk memastikan commit tercatat
