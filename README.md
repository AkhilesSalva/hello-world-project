# Hello World Project - Pekan 9

Project sederhana Hello World menggunakan HTML, CSS, dan JavaScript.

## Deskripsi
Project ini dibuat sebagai tugas Pekan 9 untuk mendemonstrasikan pembuatan source code, penggunaan Git, dan push ke GitHub.

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript (ES6)

## Cara Menjalankan
1. Buka file `index.html` di browser
2. Atau gunakan Live Server extension di VS Code

## Fitur
- Tampilan Hello World dengan desain modern
- Animasi gradient pada heading
- Menampilkan waktu saat ini
- Interaktif - klik heading untuk mengubah warna
